# CoW Protocol - Website
This is the website for CoW Protocol 

## Setup
```bash
# Install dependencies
yarn
```

## Dev server
```bash
# Run server
yarn dev
```

## Lint
```bash
# Run eslint
yarn lint
```


## Build
```bash
yarn build
```

# Run server
```bash
# Run app
yarn start
```

# Internationalization
```bash
# Extract internationalization literals from the code
yarn i18n:extract

# Generate JS files for the language file
yarn i18n:compile
```
